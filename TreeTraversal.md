## Depth First Traversal
================================

### In-order Traversal

* visit the left tree
* visit the root node
* visit the right tree



### Pre-order Traversal

* visit root node 
* visit left tree
* visit right tree


### Post-order Traversal

* visit right tree
* visit root node
* visit left tree
